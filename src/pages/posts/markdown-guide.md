# Markdown Guide

This post will show you how to use Markdown to format your blog posts.

## Headers

Use `#` for headers:

# H1
## H2
### H3

## Lists

Unordered lists:
- Item 1
- Item 2
- Item 3

Ordered lists:
1. First item
2. Second item
3. Third item

## Code

Inline code: `const greeting = "Hello"`

Code blocks:
```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

## Emphasis

*Italic text* or _italic text_
**Bold text** or __bold text__
